# Image-Scraper - Morimens-Wiki
This tools is specifically designed to scrape specific resources from the chinese wikipedia of this game for my own english wiki project.

The tool downloads relevant game assets, translates their name into english before saving, and organizing them into translated folder names as well.

I made this tool because finding the game assets was more difficult than I anticipated and I lack the experience to decrypt their game files, but these people managed to figure it out. I doubt they would lend their data to me since I'm making a wiki too so I just scraped it. It is what it is.

## Installation
- Create a virtual enviroment
- Install requirements
- Run scraper_app.py

## Usage
- Click individual pages to scrape or scrape all
